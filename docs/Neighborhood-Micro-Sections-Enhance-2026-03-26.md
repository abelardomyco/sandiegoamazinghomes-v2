# Neighborhood micro-sections — real-world detail pass (2026-03-26)

## What changed

All **27** neighborhood markdown files under `content/neighborhoods/*.md` now have **enriched** copy for the six micro-sections (structure was already present from earlier work; this pass adds specificity and local references):

1. **What you're close to** — 3–5 short lines with named places, corridors, or landmarks (proximity).
2. **Where people go from here** — 2–3 lines on commute and common trips.
3. **Daily convenience** — 2–3 lines on errands, driving, walkability.
4. **Weekend pattern** — 2–3 lines on typical weekends.
5. **Hidden reality** — 2–3 grounded observations (honest, not negative).
6. **Trade-up / trade-down** — 2–3 lines on common move-in / move-out patterns.

## What we did *not* change

- Intro paragraphs, **The Feel** / **The vibe**, **What life looks like here**, **Housing Reality**, **Who It's For**, **Tradeoffs**, **Local Insight**, **Local favorites** / **Getting around** (older file shapes), **Rosamelia notes**, or **Internal Links** — left as-is except where a file had no prior micro-blocks (none missing).

## Style

- Sections kept **distinct**; avoided generic filler phrasing.
- Named roads, venues, and sub-areas where they help buyers orient (e.g. Cedros, Rohr Park, Snapdragon, Convoy).
