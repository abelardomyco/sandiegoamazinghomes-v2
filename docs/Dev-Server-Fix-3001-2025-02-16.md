# Dev server fix — port 3001 loading (2025-02-16)

## What was wrong

- **500 on http://localhost:3001:** Next.js failed with `Cannot find module '@tailwindcss/typography'` when compiling `globals.css`. Webpack cache (`.next`) was also corrupted ("incorrect header check").

## What was done

1. **Stopped all Node processes** so port 3001 was free.
2. **Removed `.next`** to clear the bad webpack cache.
3. **Ran `npm install`** so `@tailwindcss/typography` (and any missing deps) were installed.
4. **Started dev server:** `npm run dev` → http://localhost:3001.

## Verification

- `GET /` → 200
- `GET /neighborhoods` → 200
- `GET /matchmaker` → 200
- `GET /newsletter` → 200

## If it breaks again

1. Stop dev server (or `taskkill /F /IM node.exe` on Windows).
2. From project root: `Remove-Item -Recurse -Force .next` (if it exists), then `npm install`.
3. Start again: `npm run dev`. Site should be at http://localhost:3001.

## Dependencies

- `@tailwindcss/typography` is in `devDependencies` and required by `tailwind.config.js` for `prose` styles. Do not remove it without updating the config.
